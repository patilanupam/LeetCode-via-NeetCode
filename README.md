# 30-Day LeetCode Sprint: The Interview Forge

Welcome to your coding training arc.
For the next 4 weeks, this repo becomes your daily arena where patterns become instincts.

## Mission Rules

- Think before typing: explain your approach out loud first
- Pattern over memorization
- Consistency over intensity
- Time-box hard, reflect fast

## Weekly Cadence

- Mon-Fri: 3 problems/day (60 min)
- Sat-Sun: 5 problems/day (90 min)
- Daily focus: explain the approach out loud before coding
